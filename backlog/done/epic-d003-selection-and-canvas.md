# Selection And Canvas

- Canonical IDs: `D3`
- Status: done
- Summary or outcome: shipped the baseline selection engine and precision canvas aids.

## Scope

- Selection engine baseline with marching ants, combine modes, invert, deselect, select all, clipping, and undo or redo preservation.
- Elliptical marquee selection.
- Lasso and polygonal lasso selection.
- Magic wand selection.
- Select by color range.
- Edge refinement.
- Quick mask mode.
- Rulers, guides, and snapping.
- Grid system with persistence and snap-to-grid.
- Pixel editing mode with nearest-neighbor rendering and pixel grid overlays.

## Acceptance Criteria

- Selection and canvas precision workflows are shipped at MVP or baseline level.
