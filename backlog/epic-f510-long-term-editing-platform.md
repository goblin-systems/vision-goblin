# Long-Term Editing Platform

- Canonical IDs: `F5.1`, `F5.2`, `F5.3`, `F5.4`, `F5.5`
- Status: future
- Summary or outcome: build long-term differentiators that make Vision Goblin more continuous, extensible, and workflow-aware than a basic desktop editor.

## Scope

- `F5.1` Hybrid manual and AI editing continuity.
- `F5.2` Style memory.
- `F5.3` Context-aware export.
- `F5.4` Plugin system foundation.
- `F5.5` Workflow marketplace or internal library.

## Acceptance Criteria

- AI outputs feel like normal editable editor artifacts rather than a separate mode.
- Users can save and reapply looks or workflow assets where supported.
- Export can evolve toward destination-aware outcomes instead of raw settings alone.
- The product has a credible path to safe extension points and a discoverable workflow library.
